# Lab Activity 1: Software Relevant Tools, Standards, and Code Versioning using GitHub

## Description
This repository contains a Python application that performs three basic operations which are adding, subtracting and multiplying. It seperates the python file from src folder which has the main logic from the test cases which are in the folder tests.

## Environment
In this lab, we can observe how to create a Python workspace. The steps are as follows:

```bash
# 1. Create the directory.
mkdir lab1
cd lab1

# 2. Create and activate a Python virtual environment.
python3 -m venv .venv
source .venv/bin/activate

# 3. Initializes the Git repository.
git init

# 4. Create the directory structure.
mkdir src tests

# 5. Visual Studio Code is opened with this command.
code .
